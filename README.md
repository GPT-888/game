### 基于HTML+CSS+JS的小游戏集
这是一个基于Phaser的小游戏集，目的是方便大家学习前端的内容。大家也可以访问我的github官方网站：https://kyst.null.fit/ 或者 https://kejiyuzhe.chgskj.cn/

### 源码
每一个游戏的源码都在对应的文件夹中，如 fruit - 切水果 ，源码在fruit文件夹中
